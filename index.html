Digital Windex (ongoing)

I conceptualized and proposed the idea for Digital Windex, which was selected among a group of student submissions and presentations. With my group, we are developing a user-friendly all-in-one Windows maintenance tool to automate system diagnostics, repair, and optimizations. We use Visual Studio 2022, including C#, .NET, and WPF, focusing on a jargon-free, modular design for non-technical users. We plan to consolidate existing Windows tools including DISM, Windows Memory Diagnostics, Windows Update, and more in the program.
